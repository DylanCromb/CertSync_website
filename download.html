How to install the Android APK

Android may ask for permission before installing apps downloaded outside the Play Store. The wording varies slightly by device, but the steps are usually the same.

  1. 1

    Download the APK

    Tap the Android APK button above. If your browser asks you to confirm the file download, choose Download or OK.

  2. 2

    Open the downloaded file

    When the download finishes, tap Open. You can also find the file in your Downloads app or your browser's downloads list.

  3. 3

    Allow installation from this source

    If prompted, open Settings and enable Allow from this source for your browser or file manager. Return to the APK after enabling it.

  4. 4

    Install CertSync

    Tap Install, wait for the installation to finish, then open CertSync and sign in with your account details.

  5. 5

    Optional security step

    After CertSync is installed, you can turn off Allow from this source in Android settings.

For safety, only install the CertSync APK from this official CertSync download page. If Android warns that the app is from an unknown source, that is expected for direct APK installs.
